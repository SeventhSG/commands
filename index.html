studyseventh - Twitch Bot

Custom bot for studytimemd to boost productivity with style 🧠✨

📌 Core & Utility Commands

!commands - List all available viewer commands.
!sessionstate - Check current work/break state and duration.
!botstate - Show current bot state (work/break, talk freely, etc).
!time - Show the current time and date.
!envtest - Check if .env config is working.

📚 Study & Productivity

!work - Manually start work time (mods/streamer only).
!break - Manually start break time (mods/streamer only).
!superfocus on/off - Toggle focus mode for yourself.
!endpomo - End all focus modes for users.
!recap - Show a recap of the current stream session.
!timeline - Show a summary of today’s stream activity.

🤖 AI & Fun

!s [message] - Chat with the AI.
!studybuddy [question] - Get study tips or motivation.
!vibe - Show the current stream vibe as emojis.
!poemrecap - Generate a short poem or haiku.
!emotecombo - Suggest an emote combo.
!remindme [minutes] [msg] - Set a reminder.
!funfact - Share a random fun fact.
!dance - Send DinoDance emotes.

🎥 Stream & Vision

!analyze - Stream description + mood analysis.
!customanalyze [prompt] - Analyze stream with a custom prompt.
!streamscene - Get current stream scene details.
!analyzestate - Check analyzer’s state.

🧠 Memory & Personalization

!memory - Recall your personal memory.
!remember that ... - Save info to memory.

🎙️ Voice & Audio

!hear - Capture 15s of stream and transcribe it.
!voicestatus - Check voice recognition status.

🌍 Translation & Moderation

!translate on/off - Toggle chat translation (mods/streamer only).
!onlyonline on/off - Bot works only when live (mods/streamer only).
!talkfreely on/off - Toggle free talk mode (mods/streamer only).

📬 Feedback & Special

!feedback [message] - Submit feedback about the bot.
!special - Special appreciation command for Ioana ❤️
