StudySeventh Bot Commands

Basic Commands

!commands — List all available viewer commands.
!sessionstate — Check current work/break state and duration.
!botstate — Show current bot state (work/break, talk freely, etc).
!time — Show the current time and date.
!vibe — Show the current stream vibe as an emoji cloud.
!funfact — Share a random fun fact or trivia.
!dance — Send DinoDance emote spam to chat.

Study & Productivity

!work — Manually start work time (mods/streamer only).
!break — Manually start break time (mods/streamer only).
!superfocus on/off — Toggle focus mode for yourself.
!recap — Show a recap of the current stream session.
!timeline — Show a summary of today’s stream activity timeline.

AI & Motivation

!s [message] — Chat with the AI.
!studybuddy [question] — Ask the AI for a study tip, motivation, or explanation.
!poemrecap — Generate a short poem or haiku summarizing the stream.
!emotecombo — Suggest a custom emote combo for the current mood.
!remindme [minutes] [message] — Set a reminder for yourself.

Stream & Vision

!analyze — Get a stream description (mood/activity analysis).
!customanalyze [prompt] — Analyze stream with a custom prompt.
!streamscene — Get a detailed current stream scene description.
!analyzestate — Check the analyzer’s current state.

Memory & Personalization

!memory — Recall your personal memory.
!remember that ... — Save information (personal or general memory).

Voice & Audio

!hear — Capture 15 seconds of stream audio and transcribe it.

Feedback

!feedback [message] — Submit feedback about the bot.
